### 3. **日志管理功能**

**新增服务类：** `Services/LogManager.cs`

**功能：**
- ✅ 自动创建日志文件（按时间戳）
- ✅ 记录所有操作（添加文件、删除节点等）
- ✅ 记录编译日志（生成 CHM 的详细过程）
- ✅ 记录状态变化（左下角状态栏的内容）
- ✅ 记录异常信息（包含完整堆栈跟踪）
- ✅ 自动清理旧日志（保留 30 天）
- ✅ 一键打开日志文件夹

**使用方法：**
- **查看日志：** 点击左下角状态栏的"📁 日志"
- **日志位置：** 程序目录\logs\
- **日志格式：** UTF-8 编码的文本文件

**API：**
```csharp
// 记录操作日志
LogManager.Instance.WriteOperation("添加文件: test.html");

// 记录编译日志
LogManager.Instance.WriteCompile("=== 开始生成 CHM ===");

// 记录状态变化
LogManager.Instance.WriteStatus("编译成功");

// 记录异常
LogManager.Instance.WriteException(exception);

// 打开日志目录
LogManager.Instance.OpenLogDirectory();

// 清理旧日志（保留指定天数）
LogManager.Instance.CleanOldLogs(30);
```

## 📋 使用指南

### 查看编译日志（界面内）
1. 点击主界面底部的"编译日志"展开器
2. 查看实时的编译过程信息

### 查看日志文件（永久保存）
1. 点击左下角状态栏的"📁 日志"
2. 在打开的文件夹中查看日志文件
3. 操作日志：`operation_*.txt`
4. 编译日志：`compile_*.txt`

### 配置外部工具
1. 点击顶部工具栏的"⚙ 设置"按钮
2. 勾选要启用的外部工具
3. 设置工具路径
4. 点击"保存"

### 生成 CHM
1. 添加文件或文件夹
2. 点击"🔨 生成 CHM"
3. 选择输出目录
4. 查看编译日志确认每一步
5. 如果失败，查看日志文件获取详细信息
